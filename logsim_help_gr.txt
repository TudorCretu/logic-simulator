Υποστηριζόμενα ονόματα τύπων και ονόματα θυρών συσκευών:

AND (πύλη AND): απαιτεί μια / παράμετρο που δείχνει πόσες εισόδους έχει.
Αυτή η παράμετρος είναι ένας ακέραιος από 1 έως 16.
Για πύλη AND gate GATE:
Η παραγωγή του είναι «GATE». Οι εισροές του μπορούν να είναι GATE.I1, GATE.I2, ...

NAND (πύλη NAND): απαιτεί μια / παράμετρο που δείχνει πόσες είσοδοι έχει.
Αυτή η παράμετρος είναι ένας ακέραιος από 1 έως 16.
Για πύλη NAND GATE:
Η παραγωγή του είναι GATE. Οι εισροές του μπορούν να είναι GATE.I1, GATE.I2, ...

OR (πύλη OR): απαιτεί μια / παράμετρο που δείχνει πόσες είσοδοι έχει.
Αυτή η παράμετρος είναι ένας ακέραιος από 1 έως 16.
Για πύλη OR 'GATE':
Η παραγωγή του είναι GATE. Οι εισροές του μπορούν να είναι GATE.I1, GATE.I2, ...

NOR (NOR gate): απαιτεί μια / παράμετρο που δείχνει πόσες εισόδους έχει.
Αυτή η παράμετρος είναι ένας ακέραιος από 1 έως 16.
Για μια πύλη NOR GATE:
Η παραγωγή του είναι GATE. Οι εισροές του μπορούν να είναι GATE.I1, GATE.I2, ...

ΔΙΑΚΟΠΤΗΣ (διακόπτης): απαιτεί ένα / παράμετρο που να δείχνει την αρχική του κατάσταση (0 ή 1).

CLOCK (ρολόι): απαιτεί μια / παραμέτρους που υποδεικνύουν τον αριθμό κύκλων προσομοίωσης σε μισή χρονική περίοδο.

SIGGEN (γεννήτρια σημάτων): απαιτεί μια / παράμετρο που υποδεικνύει το πρότυπο σήματος που παράγει.
Μπορεί να υπάρχουν μόνο 0 και 1 στη συμβολοσειρά παραμέτρων.

XOR (πύλη XOR): Δεν χρειάζεται καμία παράμετρος.
Για πύλη XOR GATE:
Η παραγωγή του είναι GATE. Οι 2 είσοδοί του είναι GATE.I1, GATE.I2, ...

DTYPE (flip-flop τύπου D): Δεν χρειάζεται καμία παράμετρος.
Για ένα D-flip-flop DFF:
Οι 2 εξόδους του είναι DFF.Q και DFF.Q_BAR.
Οι 4 εισόδους είναι DFF.CLK, DFF.SET, DFF.CLEAR, DFF.DATA.
